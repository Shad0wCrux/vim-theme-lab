vim-theme-lab

A visual Vim colorscheme designer

vim-theme-lab is an interactive tool for visually designing Vim/Neovim colorschemes.


Features:
- Live visual editor
- Preset themes
- 100% client-side, no backend, no database, no tracking


Check out [vim-theme-lab](https://shad0wcrux.github.io/vim-theme-lab/)
